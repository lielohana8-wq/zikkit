# מדריך בדיקת AI Bot

## אופציה 1: סימולטור (בלי Twilio, בלי עלויות)

### דרישות
- `ANTHROPIC_API_KEY` בקובץ `.env.local`
- אם אין מפתח — הבוט יעבוד במצב דמו בסיסי

### שלבים
1. הרץ את המערכת: `npm run dev`
2. התחבר כבעל עסק
3. לך ל-**AI Voice Bot** > טאב **Simulator** (🧪)
4. כתוב הודעה כאילו אתה לקוח שמתקשר:
   - "שלום, המזגן שלי לא עובד"
   - "יש לי נזילה במטבח"
5. הבוט ישאל שאלות — שם, כתובת, תיאור הבעיה, דחיפות
6. כשהבוט אוסף את כל הפרטים — **ליד אמיתי נוצר ב-Firestore**
7. תראה באנר ירוק "ליד נוצר" עם מספר הליד
8. לך לדף **Leads** — הליד יופיע שם עם תמליל מלא

### מה הסימולטור בודק
- אותו Prompt בדיוק כמו הבוט האמיתי
- אותו קוד יצירת ליד
- אותה לוגיקה של זיהוי דחיפות
- אותם Flow rules
